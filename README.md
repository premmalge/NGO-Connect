NGO Connect

NGO Connect is a Java-based web application developed to connect volunteers with NGOs and simplify NGO management through an admin verification system.



 Features

 User Module

* User Registration & Login
* Explore NGOs
* Search NGOs
* Filter NGOs by Category
* Send Volunteer Requests
* Track Volunteer Request Status

 Admin Module

* Admin Login
* Add/View NGOs
* Approve or Reject NGOs
* Manage Volunteer Requests
* Dashboard Statistics

 NGO Registration Module

* NGOs can send registration requests
* Admin verifies NGOs before public visibility
* Only approved NGOs appear to users



 Tech Stack

* Java
* JSP & Servlets
* JDBC
* MySQL
* HTML/CSS
* Apache Tomcat
* Git & GitHub



 Database Tables

* users
* ngos
* volunteer_requests
* ngo_requests
* admin



 Project Workflow


User -> Explore NGOs -> Send Volunteer Request

NGO -> Submit Registration Request

Admin -> Verify NGO -> Approved NGOs visible to users
